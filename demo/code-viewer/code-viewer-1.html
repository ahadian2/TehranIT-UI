آموزش استفاده از Code Viewer

اضافه کردن فایل‌ها اجباری

1 - استفاده از Sass

اگر از Sass استفاده می‌کنید، فایل code-viewer.scss را به پروژه‌ی اصلی‌تان ایمپورت کنید تا رنگ‌ها و استایل‌ها را سفارشی کنید. این فایل در مسیر زیر میباشد.

برای ایمپورت باید به فایل app.scss در آدرس assets/scss/app.scss بروید و در محل مشخص شده یعنی "ایمپورت کامپوتت ها"، این کار را انجام دهید.

2 - افزودن فایل js

این کامپوننت نیاز به فایل Js خود دارد. میتوانید محتویات آن را به فایل Jquery خود اضافه کنید و یا به صورت مستقل به پروژه خود اضافه کنید. مسیر این فایل در زیر آمده است.

نحوه استفاده از Code Viewer

برای استفاده از Code Viewer با توجه به نوع زبان برنامه نویسی میتوانید یکی از مدل های زیر را انتخاب کنید و در کد های خود استفاده کنید. Code Viewer در حال حاضر از زبان های html, css, js, jquery, csharp پشتیبانی میکند.

توجه : باید مقدار scrip در کد به script تغییر کند.

تغییر رنگ‌ها

1 - روش Sass (توصیه‌شده)

داخل _code-viewer.scss متغیرهای رنگی را تغییر دهید. سپس فایل Sass را کامپایل کنید.

2 - روش CSS Override (سریع)

در یک فایل CSS جدا یا داخل تگ style از کد های زیر استفاده کنید. این روش سریع است ولی تغییرات روی Sass دائمی‌تر و تمیزتر است.
